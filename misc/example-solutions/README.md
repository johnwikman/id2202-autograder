# Example Solutions

This directory contains solutions for the example grading tags under
`example/tests` (seen from the root of the repository). These can be copied to
other repositories for testing:

```sh
export MY_TEST_REPO="path-to-repository"
mkdir -p ${MY_TEST_REPO}/solutions
cp -rT hello       ${MY_TEST_REPO}/solutions/hello
cp -rT hello-asm   ${MY_TEST_REPO}/solutions/hello-asm
cp -rT hello-extra ${MY_TEST_REPO}/solutions/hello-extra
cp -rT hello-file  ${MY_TEST_REPO}/solutions/hello-file
```
